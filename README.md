#  Atividade API -- Controle  de  Produtos e Estoque 
## Função 
- Adicionar produtos
- Listar todos
- Atualizar preço ou quantidade
- Excluir produto
- Buscar produtos da tabela

# Como usar 
## DOCS 
entre no .env
- Instale os requirements.txt no terminal bahs

pip install dotenv fastapi psycopg2-binary psycopg2 uvicorn

entre na pasta back-end
- rode o main.py no terminal

uvicorn main:app --reload

## SITE STREAMLIT
entre na pasta front-end
-rode o app.py no terminal

python -m streamlit run app.py
